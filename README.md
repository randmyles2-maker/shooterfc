ShooterFC
ShooterFC is a high-octane First-Person Shooter and Driving Simulator engineered with Three.js. This technical showcase pushes the boundaries of browser-based WebGL, merging expansive, optimized environments with sophisticated vehicle mechanics and visceral combat.

⚡ Technical Core & Features
Dynamic Vehicle Engineering: Command a highly detailed golf cart featuring reactive steering and procedurally animated hand-joints that realistically grip the wheel during maneuvers.

High-Fidelity Grassland: Experience a lush, "UHD" environment powered by 15,000+ instanced grass blades, maintaining silky-smooth performance through advanced GPU draw-call optimization.

Tactical Arsenal: Master five distinct weapon platforms—Assault Rifle, Shotgun, Sniper, SMG, and Pistol—each with unique recoil profiles and visual feedback.

Atmospheric Excellence: Powered by ACES Filmic Tone Mapping, dynamic depth-based fog, and custom motion blur for a true cinematic feel.

Impactful VFX: Reactive enemy knockback physics paired with a custom blood-splatter particle system for satisfying combat loops.

🕹️ Command & Control
Tactical Movement: W, A, S, D

Combat Engagement: Left Click

Vehicle Interaction: E (Approach the cart to drive)

System Menu: ESC

🛠️ Architecture & Performance
ShooterFC is built on a custom-designed engine framework (game-engine.js) focused on stability and speed:

Instanced Rendering: Minimizes overhead by rendering massive foliage counts in a single pass.

Responsive Physics: Custom velocity and recoil algorithms provide a "snappy" arcade-shooter feel without the weight of external libraries.

Adaptive Visuals: High-precision rendering with user-toggleable motion blur to balance aesthetics and hardware performance.
